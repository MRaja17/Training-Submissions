# Pandas Fundamentals — Challenge Submissions

This folder contains completed materials for all challenge levels:

- **Easy 1** — `easy1_concepts.md`
- **Easy 2** — `easy2_toy_example.ipynb`
- **Intermediate 1** — `intermediate1_real_dataset.ipynb` (uses `data_iris.csv`)
- **Intermediate 2** — `intermediate2_with_sklearn.ipynb`
- **Hard 1** — `hard1_optimization.ipynb`
- **Hard 2** — `hard2_mini_project/` (includes `sales.csv`, `mini_project.ipynb`, and generated outputs)

## Quick Start
Open each notebook in Jupyter or VS Code, run all cells top‑to‑bottom.

## GitHub Submission (Command Line)
From a local clone of your repo (e.g., `Training-Submissions`):

```bash
# 1) Create a feature branch
git checkout -b feature/pandas-fundamentals

# 2) Copy this folder into your repo root (or under a subfolder like data_science/)
# (Example assumes this folder is on your Desktop; adjust path as needed)
cp -R ~/Desktop/pandas_fundamentals_challenges ./pandas_fundamentals_challenges

# 3) Add, commit, push
git add pandas_fundamentals_challenges
git commit -m "Add Pandas Fundamentals challenge solutions"
git push -u origin feature/pandas-fundamentals

# 4) On GitHub: Open a Pull Request from 'feature/pandas-fundamentals' → 'main'
```

## GitHub Submission (Web UI only)
1. In your repository, click **Add file → Upload files**.
2. Drag‑drop the entire `pandas_fundamentals_challenges` folder (or upload files inside it).
3. In the **Commit changes** box, choose **Create a new branch** → name it `feature/pandas-fundamentals`.
4. Click **Propose changes** → **Create pull request**.
